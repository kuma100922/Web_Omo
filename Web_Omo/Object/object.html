おもしろプロジェクト目標

私たちは、"ロボットとの共生を目指すグループ"です。

現在ロボットやプログラミングなどの科学技術分野は瞬く間に世の中に浸透しています。

私たちは、科学技術とこれからの未来を担う子供たちとの懸け橋になりたいと思っています。

ロボットに興味を持っていただくことはもちろん、私たちも楽しみながらこの活動を通して技術の向上に努めます。

習得したい技術

ソフトウェア分野

・AI（人工知能）
おもしろプロジェクトの目標であるロボット制作にはAIの技術を盛り込み、人間が操作するロボットとAIによる自動制御のロボットを使用し、何らかのゲーム性を確立することを考えています。

・Webサイト制作
おもしろプロジェクトの活動内容の報告のため、Webサイトをチームで管理します。バージョン管理にはGitと呼ばれるバージョン管理プラットフォームを使います。

・AR（拡張現実）
スマホなど電子機器を通して見ると本来存在しないはずの物体が現れる技術です。また、3Dゴーグルなどで、ロボットの視点を写し、まるで自分自身がロボットになったかのような体験もできます。上記のAIロボットを使用したゲームと組み合わせることでさらなるクリエイティブ性が追求できると考えます。

・組み込み系プログラミング
マイコンなどの特定のハードウェアを動かすためのプログラミングを「組み込み系プログラミング」と言います。マイコンとは、電子制御を行うための部品であり、マイコンはそのコンパクトさから、多岐に渡って使用されています。

今回使用したマイコン

  • Jetson Nano
  • Raspberry Pi
  • Arduino nano
